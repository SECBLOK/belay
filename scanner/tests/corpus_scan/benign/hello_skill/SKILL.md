# Hello World Skill

This skill prints a greeting message.

## Overview

The hello-world skill demonstrates the basic skill structure without any
security-sensitive operations.

## Usage

Simply invoke the skill and it will greet the user by name.

## Examples

- "Say hello to Alice" → "Hello, Alice!"
- "Greet Bob" → "Hello, Bob!"

## Notes

This skill requires no external tools or permissions.
